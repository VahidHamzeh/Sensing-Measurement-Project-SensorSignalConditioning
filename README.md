# Sensor Signal Conditioning & Measurement Project

This repository contains the design, simulation, and implementation files for the **Sensors and Measurement Systems** course project at **Sharif University of Technology** (Fall 2025).

## 📌 Project Overview
The objective of this project was to design and simulate an **Analog Front-End (AFE)** circuit capable of conditioning signals from a specific transducer/sensor. Given the low amplitude of sensor signals, the design focuses on:
- **Amplification:** Scaling the signal to the required range.
- **Filtering:** Removing noise and interference to improve the Signal-to-Noise Ratio (SNR).
- **Simulation:** Using professional circuit simulation software (e.g., LTSpice/PSpice) to validate performance.

## 🛠️ Key Features
- **Signal Processing:** Design of instrumentation amplifiers and active filters tailored to the sensor's frequency and amplitude specifications.
- **Circuit Simulation:** Comprehensive validation of the design under various environmental conditions.
- **Data Conversion (Bonus):** Implementation of ADC (Analog-to-Digital) and DAC (Digital-to-Analog) conversion modules.
- **Noise Analysis:** Investigation and mitigation of common-mode noise and environmental interference.

## 📁 Repository Structure
- `/Circuit-Designs`: LTSpice/PSpice schematics and simulation files.
- `/Report`: The final technical report (PDF) detailing the methodology, calculations, and performance analysis.
- `/Presentation`: The final PowerPoint presentation.
- `/Data`: Raw data and simulation output plots.

## 🚀 Tools & Technologies
- **Simulation:** LTSpice / PSpice
- **Analysis:** MATLAB / Python (for signal verification)
- **Documentation:** LaTeX / Microsoft Word
